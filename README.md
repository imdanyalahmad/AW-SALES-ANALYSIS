# AW-SALES-ANALYSIS - Power BI Report
 

## Project Overview
This Report analyzes Sales data to understand and track Business performace . The Report is created using Microsoft Power BI.

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- Data Visualization

## Dataset
The dataset contains Sales records including:
- customers_lookup
- product_categories_lookup
- product_Subcategoreis_lookup
- products_Lookup
- Returns Data
- Sales Data
- Territoreis_lookup. 

## Dashboard Features

### Executive Summary
Provides a high-level overview of business performance.

Key Metrics:
- Total Revenue: $24.91M
- Total Profit: $10.46M
- Total Orders: 25K
- Return Rate: 2.17%

Insights include:
- Monthly revenue trend
- Orders by continent
- Product performance
- Revenue, cost, and profit trends

### MAP
 Continents on Map showing Contries with Total Orders,Total Cost,Total Profit,Total Revenue.
### Product Performance
Analyzes product sales and return patterns.

Key Metrics:
- Monthly Revenue
- Monthly Orders
- Monthly Returns

Insights include:
- Weekly revenue trend
- Weekly returns trend
- Top performing products
- Product return patterns
### Customer Analysis
Analyzes customer purchasing behavior.

Key Visuals:
- Top 100 customers by revenue
- Orders by gender
- Orders by income level
- Orders by occupation
- Orders by age
- Monthly orders and revenue trends

This dashboard helps identify high-value customers and customer segments.

## Key Insights
- A small group of customers contributes a large portion of revenue.
- Certain products generate higher profits than others.
- Return rates vary across product categories.
- Revenue trends show consistent growth over time.
